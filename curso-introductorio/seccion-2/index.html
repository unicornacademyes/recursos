Leccion 2.1 · 5 min

IA, big data y analytics.
No son lo mismo.

Estos tres terminos aparecen juntos constantemente pero describen cosas distintas. Entender la diferencia te permite hablar con precision en cualquier conversacion del sector.

Big Data
El volumen que lo cambio todo
Datos masivos que no caben en un Excel
Definido por las 3V: Volumen, Variedad, Velocidad
Requiere infraestructura especial para almacenar y procesar
No es un producto ni una herramienta. Es un fenomeno.
Analytics
El proceso de extraer valor
Convertir datos en informacion util
Responde preguntas de negocio concretas
Va de descriptivo a prescriptivo segun la complejidad
Es el trabajo central del analista de datos
Inteligencia Artificial
El motor que aprende solo
Sistemas que aprenden patrones de los datos
Incluye machine learning, deep learning y LLMs
Automatiza decisiones y genera predicciones
Necesita datos de calidad para funcionar bien
💡

Una forma de verlo: Big Data es el ingrediente. Analytics es la cocina. La IA es el chef que aprende a cocinar solo. El analista es quien decide que plato preparar y para quien.

Dentro de la IA hay ramas que vale la pena conocer por nombre:

🧠Machine LearningDENTRO DE IA

Algoritmos que aprenden de datos historicos sin ser programados explicitamente. El modelo descubre patrones solo. Ejemplo: predecir que clientes van a abandonar un servicio.

💬LLM · Large Language ModelDETRAS DE CHATGPT

Modelos entrenados con enormes cantidades de texto que pueden leer, escribir, resumir y generar contenido en lenguaje natural. ChatGPT, Claude y Gemini son LLMs.

🔮Deep LearningDENTRO DE ML

Redes neuronales artificiales que imitan el cerebro humano. Base de reconocimiento de imagenes, voz y los modelos de lenguaje mas avanzados.

Leccion 2.5 · 5 min

Las palabras que
todos usan y
pocos entienden.

Este es el vocabulario esencial del sector. No hace falta memorizarlo todo hoy — lo vas a ir incorporando naturalmente. Pero tenerlo claro desde el principio te ahorra mucha confusion.

DatasetFUNDAMENTAL

Un conjunto de datos organizado en filas y columnas. Es la materia prima de cualquier analisis. Sin dataset, no hay nada que analizar.

Una tabla de Excel con 10.000 filas de ventas, cada una con fecha, producto, cantidad y precio es un dataset.
APICONECTA SISTEMAS

Application Programming Interface. Un puente que permite que dos sistemas se comuniquen entre si e intercambien datos de forma automatica.

Cuando una app del tiempo muestra el pronostico, lo obtiene llamando a una API meteorologica. Cuando usas ChatGPT desde otra herramienta, se conecta via API de OpenAI.
ModeloCLAVE EN IA

El resultado de entrenar una IA con datos. Es el sistema que aprendio los patrones y que ahora puede hacer predicciones o generar respuestas.

GPT-4 es un modelo de lenguaje. Un modelo de fraude bancario detecta transacciones sospechosas. Cada uno fue entrenado con datos distintos para un objetivo distinto.
PipelineFLUJO DE DATOS

Un proceso automatizado que mueve y transforma datos de un lugar a otro. Como una tuberia: los datos entran por un lado, se procesan en el camino y salen listos para analizar.

Un pipeline puede extraer ventas del ERP cada noche, limpiarlas y cargarlas en un dashboard que el equipo ve a la manana siguiente.
ETLPROCESO CLASICO

Extract, Transform, Load. El proceso de extraer datos de sus fuentes, transformarlos y limpiarlos, y cargarlos en un sistema listo para analizar. Es el trabajo invisible que hace posible todo lo demas.

Un analista hace ETL cuando descarga datos del CRM, los limpia en Excel eliminando duplicados y los carga en Power BI para crear un dashboard.
Cloud · NubeDONDE VIVE TODO

Servidores remotos accesibles por internet donde se almacenan, procesan y analizan datos. En lugar de tener computadoras propias enormes, las empresas alquilan capacidad de computo.

AWS (Amazon), Google Cloud y Azure (Microsoft) son los tres grandes proveedores de nube. La mayoria de las empresas guardan sus datos en alguno de estos tres.
PromptINPUT DE LA IA

La instruccion o pregunta que le das a un modelo de IA. La calidad del prompt determina en gran parte la calidad de la respuesta. Es una habilidad que se aprende y que marca la diferencia.

Prompt malo: "analiza esto". Prompt bueno: "Sos un analista de ventas. Estos son los datos de ventas de marzo de una tienda de ropa. Identifica las 3 categorias con mayor caida y explica posibles causas."
KPIMEDIR EL EXITO

Key Performance Indicator. Una metrica clave que mide el rendimiento de un objetivo de negocio. Buenas decisiones requieren los KPIs correctos.

Para un ecommerce: tasa de conversion, ticket promedio, tasa de abandono de carrito. Para un SaaS: churn rate, MRR, NPS.
DashboardEL ENTREGABLE CLAVE

Panel visual interactivo que muestra las metricas clave de un negocio en tiempo real. Permite que el equipo se monitoree solo sin necesitar al analista para cada consulta.

Un dashboard de ventas muestra en una sola pantalla: ingresos del mes, comparativa con el mes anterior, top productos y rendimiento por region. Todo actualizado automaticamente.

Leccion 2.5 · 5 min

Las habilidades de datos
tienen precio
de mercado directo.

Saber analizar datos no es solo una habilidad tecnica. Es una habilidad que se monetiza de cuatro formas reales y no excluyentes entre si.

🏢En tu empresa actual

Propones mejoras con datos, reduces trabajo manual y te conviertes en la referencia analitica del equipo. Las empresas retienen y promocionan a quien les da visibilidad que antes no tenian.

Ascenso o diferencial salarial del 20 al 40%
Ejemplo real

Un responsable de operaciones automatiza el reporting mensual de su equipo y pasa de Analyst a Senior en 18 meses, sin cambiar de empresa.

🔄Cambio a perfil mixto de datos

Combinas tu experiencia de sector con habilidades analiticas y accedes a roles que antes no existian para ti: Data Analyst, BI Analyst o Analyst especializado en tu area.

Diferencial salarial del 30 al 50% sobre tu rol anterior
Ejemplo real

Profesional de marketing con 5 anos de experiencia aprende SQL y Power BI. Pasa a Marketing Data Analyst en una empresa tech. Salario +35%.

💼Freelance para pymes

Las pymes necesitan analisis pero no pueden contratar un analista a tiempo completo. Tu cubres esa necesidad con proyectos puntuales de alto valor para ellas.

500 a 3.000 euros por proyecto segun complejidad
Ejemplo real

Primer cliente: dashboard de ventas para una tienda online (800 euros). Segundo mes: automatizacion de reporting para una clinica (1.200 euros). Al tercero ya tenes referencias.

🔁Consultoria recurrente

Un cliente que te contrata mes a mes para mantener dashboards, actualizar analisis o responder preguntas de negocio con datos. La forma mas predecible de monetizar.

400 a 1.500 euros por cliente al mes
Ejemplo real

3 clientes recurrentes a 600 euros al mes = 1.800 euros mensuales predecibles. Compatible con un empleo a tiempo completo.
